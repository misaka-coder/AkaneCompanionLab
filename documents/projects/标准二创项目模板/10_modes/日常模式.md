# 日常模式

这份文档只写“日常模式”的规则。

## 1. 模式定位

这里写：

- 平时聊天的氛围
- 常用场景范围
- 常用服装范围
- 常用 BGM 范围

## 2. 允许的场景

- 可进入的 `major`
- 每个 `major` 下常用的 `minor`
- 哪些背景变体最常见

## 3. 允许的服装和表情

示例：

- 校服：normal / shy / smug / cry
- 睡衣：normal

## 4. 演出原则

- 场景切换可以自然一些
- BGM 可以轻微跟着氛围变化
- 不要频繁大跳场景

## 5. 禁止事项

- 不推进强剧情主线
- 不乱触发事件专用场景
- 不把非日常专属 BGM 当普通日常 BGM
