# Changelog

All notable public changes will be documented here.

The project uses Semantic Versioning where practical. Pre-release versions may
change APIs and local data formats without migration guarantees.

## [0.1.0-alpha.1] - Unreleased

### Added

- Initial public learning and research preview.
- FastAPI companion backend and static Web client.
- Windows-first Tauri desktop-pet prototype.
- Character workshop and creator-kit tooling.
- Optional QQ/NapCat, voice, visual context, retrieval, and local capability
  integrations.
- Public release exporter that omits private history and unverified media.

### Known Limitations

- The project is not plug-and-play and may require manual or AI-assisted setup.
- Desktop-pet support is primarily tested on Windows.
- Several integrations require separately installed tools or services.
- Public exports use placeholder art until redistributable assets are supplied.
- Some features remain partial, deferred, or experimental.
